# Metacl

DSL that generates C code for different computation platforms (pure C, OpenCL, Intel Phi)

## Installation

Add this line to your application's Gemfile:

    gem 'metacl'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install metacl
